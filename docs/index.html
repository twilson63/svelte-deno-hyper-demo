Hyper by example

hyper is a core backend service for secure, scalable applications. You can spin up a hyper backend for you application in less than a minute and immediately get access to data, cache, storage, queue and search services.

hyper by example is a collection of annotated examples for how to use hyper, and the various features it provides. While it acts as a reference, it can also be used as a guide to learn about many of the features of hyper.
