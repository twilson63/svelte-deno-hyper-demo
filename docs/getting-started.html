Getting Started

In order to get started with hyper, head to https://dashboard.hyper.io and create an account. Then create a new hyper application. Then open up a terminal and create a `.env` file and copy the connection-string from the hyper and add to the `.env` file.

HYPER=[Connection-String]

import dependencies 
              
import { connect } from 'hyper-connect'
create hyper index 
              
const hyper = connect(Deno.env.get('HYPER') as string)
get all documents of type games 
              
const games = await hyper.data.query<Game>({type: 'game'})

Run this example locally using the Deno cli 

deno run https://examples.hyper.io/getting-started.js
